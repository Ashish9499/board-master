# Board Master

A modern task/board management application built with React, TypeScript, and Tailwind CSS.

## 🚀 Project Overview

Board Master is a responsive web application that allows users to manage tasks efficiently using a clean and intuitive interface. It focuses on performance, scalability, and modern frontend best practices.

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
